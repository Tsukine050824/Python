## Hướng dẫn tạo dự án mô phỏng bằng Python

### Tải Python: https://www.python.org/downloads/

### Tải IDE (Có thể sử dụng VSCode hoặc Pycharm, tôi dùng VSCode):
- VSCode: https://code.visualstudio.com/download
- Pycharm (Có 2 bản là bản Community và bản Unified):
  - Community: https://pycharm-community-edition.en.softonic.com/
  - Professional (Unified): https://www.jetbrains.com/pycharm/downloads


### Tải extension(Nếu dùng VSCode): https://marketplace.visualstudio.com/items?itemName=ms-python.python

### Tạo file
- VSCode: Tạo bất kì 1 file với tên bất kì và để đuôi là .py
- Pycharm: Chọn Python file, xong đặt tên bất kì

### Cài môi trường(Mở màn Terminal lên và gõ): 
- VSCode(Vì VSCode không nhận lệnh pip) : python -m pip install matplotlib numpy
- Pycharm: pip install matplotlib numpy
